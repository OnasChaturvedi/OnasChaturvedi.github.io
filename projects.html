Hungrrry Frog (Scratch)

This project is a fun and interactive game created on Scratch. In this game, you control a frog that must avoid snakes and eat insects to stay alive as long as possible. The longer you stay alive, the more points you accumulate. The goal of the game is to improve your reflexes as you try to avoid the snakes while eating as many insects as possible. The game uses basic physics and collision detection, and it's a simple, yet engaging way to learn the basics of game development.

Technologies Used:
  • Scratch - For creating interactive stories and games with a drag-and-drop interface.
  • Basic Game Mechanics - Collision detection, score tracking, and random enemy generation.

Freaky Pong (Godot 4)

In this project, I created a Pong clone using Godot 4. The game includes both single-player and two-player modes. The single-player mode features an AI opponent that can be adjusted through an AI difficulty slider, allowing players to choose the difficulty of the AI. The game mechanics are simple: players control paddles on either side of the screen and try to bounce a ball past their opponent. This project helped me practice game logic, physics, and AI programming in Godot, and I also learned how to design and implement a difficulty slider to adjust the AI's speed and responsiveness.

Technologies Used:
  • Godot 4 - For game development, including physics, AI, and user input handling.
  • GDScript - Used for writing the game logic and AI behavior in Godot.
