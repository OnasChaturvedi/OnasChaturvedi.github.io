Welcome to My CS50 Week 8 Project

This project is part of the CS50 course by Harvard University. It is designed to demonstrate my understanding of web development, including HTML, CSS, and JavaScript, as well as concepts learned throughout the course.

Project Overview

This project is a personal website that showcases my work, skills, and contact information. I have created this site to better understand how web development works, from the front-end design to back-end integration. The goal of this project was to build a responsive, user-friendly website that looks good on both desktop and mobile devices. Additionally, I have used modern design principles to ensure that the site is visually appealing and easy to navigate. If you have any questions or would like to get in touch with me, feel free to reach out!

Technologies Used

For this project, I used the following technologies:
  • HTML5 - For the structure and content of the webpage
  • CSS3 - For styling the webpage and making it responsive
  • JavaScript - For adding interactive elements and enhancing user experience
  • Bootstrap - For responsive layout and design components
